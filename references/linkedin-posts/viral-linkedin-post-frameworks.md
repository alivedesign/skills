# Viral LinkedIn Post Frameworks

## The Viral Formula

1. **Pattern interrupt** - Hook that stops the scroll
2. **Emotional resonance** - Content that triggers sharing
3. **Engagement catalyst** - Reason to comment, not just like

## Key Statistics

- Contrarian views: 300% higher engagement
- Vulnerable narratives outperform promotional 4:1
- Questions boost comments by 70%
- Optimal length: 800-1,200 characters

---

## The 12 Frameworks

### 1. Contrarian Hook
**Structure:** Challenge wisdom → Present alternative → Ask for debate

### 2. Data-Driven Authority
**Structure:** Present research → List stats → Draw insight → Question

### 3. Case Study with Proof
**Structure:** Client metrics → Strategy reveal → Changes list → CTA

### 4. Step-by-Step Guide
**Structure:** Promise outcome → Numbered steps → Personal proof → Save CTA

### 5. Problem-Solution Story
**Structure:** Paint pain → Transformation → What changed → Client quote

### 6. Vulnerable Failure Story
**Structure:** Rock bottom → Specific failure → Pivot → New approach → Result

### 7. Culture Showcase
**Structure:** Specific moment → Employee spotlight → Culture principle

### 8. Industry In-Joke
**Structure:** Relatable frustration → Examples → Pivot to insight

### 9. Unpopular Opinion
**Structure:** "Unpopular opinion" → Critique → Personal exception → Reframe

### 10. Fill-in-the-Blank
**Structure:** Simple prompt → Personal example → "Your turn"

### 11. Shoutout/Recognition
**Structure:** Tag people → What they contributed → Ask for others

### 12. Values Manifesto
**Structure:** State belief → Contrast list → Personal stakes

---

## Hook Templates

**Contrarian:**
- "Everyone says [common advice]. That's wrong."
- "[Thing] is dead. No, you're just bad at it."

**Curiosity:**
- "I analyzed [number] [things]. The pattern was clear:"
- "[Number] years ago, I [low point]. Today, [transformation]."

**Story:**
- "I almost [gave up/quit/failed] [timeframe] ago."
- "[Time] ago, I made the decision that changed my life."

**Direct Value:**
- "How to [outcome] in [number] steps:"
- "[Number] things NOT to [do] in [situation]:"

---

## CTA Templates

**Questions:**
- "What's your take - [option A] or [option B]?"
- "Am I wrong about [topic]?"

**Actions:**
- "Comment '[KEYWORD]' for [resource]."
- "Save this for your next [situation]."

---

## Formatting Rules

1. One thought per line
2. Use arrows (→) for lists
3. Front-load value in hook
4. 800-1,200 characters optimal
5. End with question (70% of viral posts do this)

---

## Content Decision Matrix

| Goal | Best Framework |
|------|---------------|
| Build authority | Data-Driven, Case Study |
| Drive engagement | Fill-in-Blank, Unpopular Opinion |
| Generate leads | Step-by-Step, Teaser |
| Build community | Shoutout, Culture Showcase |
| Personal brand | Vulnerable Story, Values Manifesto |

## Viral Checklist

- [ ] Hook stops scroll in first line
- [ ] Would I engage if someone else posted?
- [ ] Specific numbers/names/examples
- [ ] Clear structure (lists, arrows)
- [ ] Emotional trigger present
- [ ] Ends with engagement catalyst
- [ ] Authentic to my voice
- [ ] Provides genuine value
